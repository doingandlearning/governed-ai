# Module 3 starter app

Checkpoint baseline for **Lab 3: Choose the Right Execution Pattern**. It contains the completed **Lab 2** governed document-extraction workflow—participants use this app when applying the execution-pattern decision framework (no new backend tasks in Module 3).

**Instructor reference (extended app):** `../../demo-app/`

## Run

```bash
cd backend
cp .env-example .env
npm install
npm run dev
curl http://localhost:3000/documents/health
```

See `backend/README.md` and `../README.md` (Module 3 lab).
