# Module 3: Workflow vs Agent Design

## Objective
Choose the right execution model for each AI feature and make explicit tool-choice decisions: deterministic workflow, bounded tools, or agentic behavior.

## Contents
- **slides.md** — Session slides.
- **TEACHING_NOTES.md** — Instructor notes.
- **demos/** — Decision framework and boundary demos.
- **exercises/** — Workflow vs agent design exercise.

## Tool-Choice Focus
- Define tool-selection criteria before implementation (reliability, latency, side-effect risk, auditability, and fallback readiness).
- Justify why each tool is included or excluded for a scenario.
- Prefer the smallest tool set that satisfies outcome requirements.

## Prerequisites
Modules 1-2.
