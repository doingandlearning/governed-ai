## What this course is

- <span class="fragment">A two-day engineering bootcamp for governed AI feature delivery.</span>
- <span class="fragment">Stack context: TanStack frontend, NestJS backend, gateway patterns.</span>
- <span class="fragment">We cover workflows, guardrails, UX, evals, and deployment governance.</span>
- <span class="fragment">You leave with reusable patterns, not just demo code.</span>

---

## What this course is not

- <span class="fragment">Not a deep dive on model internals.</span>
- <span class="fragment">Not a prompt-only workshop.</span>
- <span class="fragment">Not a chat UI tutorial.</span>
- <span class="fragment">Not one-off experimentation without delivery standards.</span>

## Outcomes by the end

You should be able to:

1. <span class="fragment">Design AI features with clear architectural boundaries.</span>
